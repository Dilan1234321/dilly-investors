# Dilly Investors Site

This repository is the standalone investor surface for `investors.hellodilly.com`.

## Vercel Setup

When creating the Vercel project:

- import the `dilly-investors` repo
- keep the framework preset as **Other**

This repo already includes its own `vercel.json` and static routes.

## Domain

Attach:

- `investors.hellodilly.com`
- `capital.hellodilly.com`
- `venture.hellodilly.com`
- `invest.hellodilly.com`

## Notes

- This is the capital narrative surface for Dilly.
- `capital.hellodilly.com`, `venture.hellodilly.com`, and `invest.hellodilly.com` should redirect here.
